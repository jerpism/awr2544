```
MCU+ SDK for AWR2544 >= 9.1.0.05
SysConfig >= 1.24.0
mmWave mcuplus SDK >= 4.6.1.02
```

## NOTE: this is very much a works-on-my-machine repo and won't build on other systems without several modifications to the project configuration.
